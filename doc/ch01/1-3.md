## Упражнение 1.3

Определите процедуру, которая принимает в качестве аргументов три числа и возвращает сумму
квадратов двух бо ́льших из них.

## Решение

```racket
(define (square x) (* x x))

(define (largest x y)
  (if (> x y) x y))

(define (sum-of-squares-of-two-largest a b c)
  (+ (square (largest a b))
     (square (largest b c))))

```
